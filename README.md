# Привычки

Мобильное приложение для отслеживания привычек на Flutter.

## Текущее состояние
- Реализована архитектура с разделением слоев: `presentation`, `domain`, `data`.
- Локальное хранение: SQLite (`sqflite` + `sqflite_common_ffi` для desktop).
- Заложены интерфейсы для будущей синхронизации с API через `RemoteDataSource`.

## Функции
- Полностью русскоязычный интерфейс.
- Главный экран: список привычек, прогресс по цели и недельный календарь с круглыми кнопками дней, выделением текущего дня и отметкой выполнения по дням.
- Отмечать привычку можно только за текущий или прошлые дни.
- Создание и редактирование привычки через отдельный экран настроек: название, 1 из 6 цветов, цель по количеству и периоду (`неделя`, `месяц`, `год`).
- Количество выполнений в цели выбирается из выпадающего списка.
- Проверка ограничений цели: не больше `7` раз в неделю, `31` в месяц, `366` в год.
- Экран привычки: вкладки `Календарь` и `Статистика`, месячный календарь с круглыми кнопками дней, переключением месяцев `<` `>`, выделением текущего дня и редактированием настроек привычки.
- В режиме редактирования можно удалить привычку.
- Карточки статистики вынесены в отдельную вкладку `Статистика`: кнопки `-`/`+`, сетка 2 карточки в ряд, максимум 20 карточек (10 рядов), выбор типа статистики внутри карточки.
- Поддерживаются типы статистики: `дней подряд`, `прогресс`, `дней всего`, `дней до конца`, `заметка`.
- Добавлены светлая и темная темы с киберпанк-оформлением и переключением прямо в приложении.
- Linux desktop запуск и Android APK сборка настроены и рабочие.

## Структура
- `lib/core` — общие утилиты.
- `lib/features/habits/domain` — сущности и интерфейсы репозиториев.
- `lib/features/habits/data` — SQLite datasource, модели, реализация репозитория.
- `lib/features/habits/presentation` — экраны, провайдеры, UI-виджеты.

## Запуск
После установки Flutter выполните:

```bash
flutter pub get
flutter run -d linux
flutter build apk
```
